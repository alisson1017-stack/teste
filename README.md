# **Projeto teste**

## O que é?

Esse projeto foi criado apenas para praticar os comandos de **controle de versão** (`add, commit, push, pull, revert, reset, etc.`). Por isso há tantos commits <u> sem alterações relevantes</u>.

Embora seja um teste para aprática de comandos de controle de versão, há um sistema funcional ao rodar o **index.html**.

Nele você achará um contador regressivo (se você enviar um número positivo) ou progressivo (se for enviado um número negativo) que vai do [número enviado] até zero. Embora ele não tenha sido muito refinado.
